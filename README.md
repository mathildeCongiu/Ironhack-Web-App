# Happy GM to you!

## Description

The idea of the project is to compare the birthday of the user with the birthdays of the worldwide Chess Grand Masters. That way you can see which grand-master you could have been born ! So close...

## MVP (DOM - CANVAS)

- Sign Up
- Sign In 
- FAQ page
- Dashboard
- Main page with API
